STEP 1 — Replit에서 가져올 정보 수집

이 파일을 Claude Code에 붙여넣고 시작하세요.
각 항목을 확인하면서 값을 채워넣으면 됩니다.


1. 환경변수 (Secrets) 전체 목록
Replit 좌측 메뉴 → Secrets 탭에서 확인
아래 항목들을 복사해서 값을 채워주세요:
# 데이터베이스
DATABASE_URL=

# Google 연동
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GOOGLE_REDIRECT_URI=

# OneDrive 연동
ONEDRIVE_CLIENT_ID=
ONEDRIVE_CLIENT_SECRET=
ONEDRIVE_REDIRECT_URI=
ONEDRIVE_TENANT_ID=

# Telegram
TELEGRAM_BOT_TOKEN=
TELEGRAM_CHAT_ID=

# 앱 설정
SESSION_SECRET=
NODE_ENV=production

# 기타 (Replit Secrets에 있는 것 모두 추가)

2. DB 백업
Replit Shell에서 아래 명령어 실행:
bash# 백업 생성
pg_dump $DATABASE_URL > backup_$(date +%Y%m%d).sql

# 파일 크기 확인 (0이면 오류)
ls -lh backup_*.sql
백업 파일을 로컬 PC에 다운로드 해두세요.
(Replit 파일 탐색기 → 파일 우클릭 → Download)

3. 현재 Replit 도메인 확인
현재 앱 주소: https://________________.replit.app
이 주소가 Google, OneDrive OAuth에 등록되어 있습니다.
새 서버로 이전 시 이 주소를 교체해야 합니다.

4. Google Cloud Console 설정 확인
https://console.cloud.google.com 접속
확인할 내용:

 프로젝트 이름: ________________
 OAuth 2.0 클라이언트 ID 확인
 승인된 redirect URI 목록 확인 (현재 Replit 주소 포함되어 있을 것)
 활성화된 API 목록:

 Gmail API
 Google Calendar API
 Google Drive API (있다면)




5. Azure (OneDrive) 설정 확인
https://portal.azure.com 접속 → 앱 등록
확인할 내용:

 앱 이름: ________________
 Application (client) ID: ________________
 Redirect URI 목록 확인 (현재 Replit 주소 포함)

Node.js 버전 (engines 필드)
